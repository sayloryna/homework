Métodos de arrays-ejercicios

1.Filtrar palabras cortas: Crea una función que reciba un listado de palabras y devuelva otro listado que contenga sólo las palabras con 3 o menos letras.

Por ejemplo: si recibe “hola”, “sal”, “no” y “cielo”, debe devolver “sal” y “no”.

const randomWords= ["casa","pedo","gato", "naranja","platano","pintura","zanahoria","pis","caca","gas","pio","ojo","judia"]

function onlyThreeLetters(listOfWords){
return listOfWords.filter((word)=>word.length ===3)
}
console.log(onlyThreeLetters(randomWords))

2.Generador de nombres hacker: Escribe una función que genere y devuelva un nombre de hacker aleatorio. Lo generará a partir de dos listas de palabras (una con sustantivos y otras con adjetivos).

Ejemplos: “SleepyDolphin”, “SadKid”, “GoodDoctor”, “MadDog”, “SneakyTiger”, etc.

    
const names=  [ "mouse","fire" , "keyboard","chair", "fingers", "glasses", "cat"];
const adjetives = ["sneaky","flaming", "fast","slow","tremendous","sloppy","old"]
function createHackerName(names,adjetives){
    return adjetives[Math.floor(Math.random()*adjetives.length)] +"-"+ names[Math.floor(Math.random()*names.length)]
}
console.log(createHackerName(names,adjetives))

3.Carrera de tortugas: Crea un array de objetos que representen tortugas con dos propiedades: nombre y velocidad.

Crea una función que reciba el array de tortugas, simule una carrera y devuelva la tortuga ganadora. Utiliza el método sort() para simular una carrera de tortugas y determinar quién es el ganador.

    
const tortugasDeCarreras= [
    {name:"paca",speed:100},
    {name:"tula", speed:200},
    {name:"fabiana",speed:50}
]

function racingTortugas(tortugas){
    return tortugas.sort((tortuA,tortuB)=>tortuB.speed - tortuA.speed)

}
console.log(racingTortugas(tortugasDeCarreras))

4.Inversor de palabras: Crea una función que reciba un array de palabras y devuelva otro array con las mismas palabras, pero invertidas. Utiliza el método map() para transformar cada palabra en una nueva palabra que contenga las letras en orden inverso.

    
const wordList = ["casa", "pendejo", "palindromo","ojo",];

function reverseWords(listOfWords) {
    return listOfWords.map(word => word.split('').reverse().join(''));
}

console.log(reverseWords(wordList));
