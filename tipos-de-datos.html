Conversor de temperatura:

  • Declara una variable para la temperatura en grados Celsius (C) e inicialízala.
  • Convierte la temperatura a Fahrenheit usando la fórmula: F = (C * 9/5) + 32. Almacénala en una variable.
  • Convierte la temperatura a Kelvin usando la fórmula: K = C + 273.15. Almacénala en una variable.
  • Imprime por consola, de una forma amigable y en un único mensaje, la temperatura en grados Celsius, Fahrenheit y Kelvin.

        
const tempCelsius = 30;
const tempFahrenheit= tempCelsius*1.8+32;
const tempKelvin = tempCelsius+273.15;

console.log(tempCelsius + "Cº equivalen a: " + tempFahrenheit + "ºF y " + tempKelvin + "ºK." )

Calculadora de horas de sueño:

  • Almacena tu edad actual en una variable.
  • Almacena tu edad máxima esperada.
  • Almacena la cantidad estimada de horas de sueño por día.
  • Calcula cuántas horas de sueño necesitarás en total para el resto de tu vida.
  • Muestra el resultado en la consola de la siguiente manera: “Necesitarás NN horas de sueño para llegar a la respetable edad de X años”.`
Nota: cuenta con que todos los años tienen 365 días y que estás ejecutando el programa justo el día de tu cumpleaños. 

        
const currentAge = 30;
const maxAge = 33
const sleepHoursPerDay = 8;
const totalSleepingTimeInLife = (maxAge - currentAge) * 365*8;
const totalAwakeTimeInLife = (maxAge - currentAge) * 365*(24-sleepHoursPerDay);

console.log("Necesitarás " + totalSleepingTimeInLife + " horas de sueño para llegar a la respetable edad de "
+ maxAge + " años. Por lo que te quedan "+totalAwakeTimeInLife + " horas para vivir.");
